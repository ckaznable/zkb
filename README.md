# Zyan-Ken-Pa

p2p paper scissors rock game in cli

---

## Usage

```shell
Usage: zkp-cli [OPTIONS]

Options:
      --host               make host
  -c, --connect <CONNECT>  target host ip [default: ]
  -h, --help               Print help
  -V, --version            Print version
```

host:
```shell
zkp-cli --host
```

guest:
```
zkp-cli -c <ip>
```
